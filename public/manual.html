System Manual

Key Features:

  1. Welcome Page:
    • It's the default welcoming page for the website. You will have three options to pick from: New User, Login, and Help.
  2. New User:
    • Enter your new username and password. Make sure that you do not use an exact username already made and in the system to void any errors and confusion. After successfully signing up, the IT department will be notified and will manually add you into the system.
  3. Login:
    • Enter your username and password. Ensure that all information is written correctly and that your password is secured.
  4. Help:
    • This is the option you will pick if you have any questions or problems with the website. You will have a selection of three options that you may select: System Manual, Program Description, and Security Protocols.
  5. System Manual:
    • This is meant to teach you the basics on what each option on each page does. Always come back to it if you're not sure what something does.
  6. Program Description:
    • Describes the general purpose for this program.
  7. Security Protocols:
    • Basic every day protocols to follow to not have any unwanted users to view sensitive information without permission.
  8. Main Menu:
    • It's the temporary default page once you have logged in. You will have a selection of two options to select from: Patient Records, and Logout.
  9. Patient Records:
    • Use to find any patient records quickly. The system will automatically fill in previously entered details of selected patients. You're able to edit said record and view the audit log.
  10. Edit:
    • After selecting a patient’s record, you have the option to update any information about the patient. Press “submit” to save changes to the database.
  11. Audit Log:
    • This lets you view who has accessed or updated records of any of the patient’s information.
  12. Add New Patient:
    • Use to access a blank form to enter the new patient’s information. Press “submit” to save it into the database.
  13. Logout:
    • Logs you out of your account and will return to the default welcoming page.
